### 1
1. copy
```
Give a derivation for

     A true
  ----------- copy
 A true ∧ A true

:
    ------    ------
    A true    A true
  ---------------------- ∧I
      A ∧ A true
```

2. copy - many
```
Written proof for

     A true
  ------------------ copy-many
 A true ∧ ... ∧ A true
: Because A true ⊢ A true, by ∧I, A true ⊢ A true ∧ C, where C itself can be
A true ∧ C. So this inductively define a sequence with base case A true
and inductive step A true ∧ C.
```
